# 配置文件中文说明

程序只读取数据，下面带下划线的 `_字段说明` 仅供查阅，可保留不影响显示。

---

## duty.json（每周安排）

| 英文字段 | 中文含义 |
|----------|----------|
| `title` | 页面标签名称 |
| `records` | 每周一条记录 |
| `time` | **时间**，`[开始日期, 结束日期]`，如 `["2026-06-01","2026-06-07"]` |
| `weekdayShare` | **周内分享**（两天内），人名数组，可多人 |
| `checkIn` | **督促打卡**负责人 |
| `checkInPublish` | **打卡任务发布**人 |
| `saturday` | **周六** |
| `saturday.sing` | 周六 **唱歌**的人 |
| `saturday.host` | 周六 **主持**的人 |
| `sunday` | **周日** |
| `sunday.pray` | 周日 **祷告**的人 |
| `sunday.share` | 周日 **分享**的人（可与祷告同时填） |
| `sunday.songs` | 周日 **歌曲**列表，一般两首 |
| `sunday.songs[].name` | **歌名** |
| `sunday.songs[].url` | 歌曲 **链接** |

**时间状态**（自动，不用手写）：当前 &lt; 开始 → 未执行；在中间 → 正在执行；&gt; 结束 → 执行完。

### 每周一条示例

```json
{
  "time": ["2026-06-01", "2026-06-07"],
  "weekdayShare": ["张三", "李四"],
  "checkIn": "王五",
  "checkInPublish": "赵六",
  "saturday": { "sing": "甲", "host": "乙" },
  "sunday": {
    "pray": "丙",
    "share": "丁",
    "songs": [
      { "name": "奇异恩典", "url": "https://..." },
      { "name": "赞美之泉", "url": "https://..." }
    ]
  }
}
```

## config.json（读经/打卡统计）

| 英文字段 | 中文含义 |
|----------|----------|
| `title` | 页面标签名称 |
| `people` | **全体成员**名单 |
| `records` | 每日打卡记录 |
| `time` | 当天 **日期** |
| `label` | 可选，日历标题额外文字 |
| `all` | 当天应参与的 **全部人员** |
| `delete` | 当天 **缺席/未打卡** 的人员 |
| `saturday` | 该条记录下 **周六参与** 人员（方块右半蓝色） |
| `sunday` | 该条记录下 **周日参与** 人员（方块右半蓝色） |

### 每条 records 示例

```json
{
  "time": "2026-06-02",
  "all": ["李新科", "宋袁峰"],
  "delete": ["宋袁峰"],
  "saturday": ["赵正杨", "李新科"],
  "sunday": ["王黎阳"]
}
```

左半绿 = 已打卡，左半红 = 缺席；若在 `saturday` 或 `sunday` 中，右半再加蓝色。
